## 🎯 作业目标

本作业的核心目标是：

1. **掌握提示词迭代技能**：通过至少5轮迭代，学习如何从失败的提示词逐步优化到优秀的提示词
2. **理解可视化设计原则**：在迭代过程中体会图表设计要素（标题、标注、配色、交互等）的重要性
3. **培养批判性思维**：学会识别可视化作品的不足并提出改进方案
4. **产出高质量可视化作品**：最终呈现一个清晰、准确、美观的气候变暖数据可视化

## 📊 数据来源

**NASA GISS Surface Temperature Analysis (GISTEMP v4)**

数据链接：https://data.giss.nasa.gov/gistemp/data_v4.html

### 推荐使用的数据文件

| 数据类型 | 文件格式 | 说明 |
|----------|----------|------|
| 全球年度平均温度异常 | CSV | 1880年至今，最常用 |
| 半球年度平均温度异常 | CSV | 南北半球对比 |
| 分区年度平均温度异常 | CSV | 不同纬度带分析 |

**数据说明**：温度异常（Temperature Anomaly）是指相对于1951-1980年平均值的偏差，正值表示变暖，负值表示变冷。

## 📝 作业要求

### 一、数据准备（10分）

- [ ] 下载并阅读原始数据文件
- [ ] 理解数据结构和字段含义
- [ ] 简要记录数据的基本信息（时间范围、数据量等）

### 二、提示词迭代过程（60分）

**核心要求：完成至少5轮迭代，每轮必须记录在模板中**

| 迭代阶段 | 要求 |
|----------|------|
| 第1轮：初始尝试 | 使用最简单的提示词，接受任何结果 |
| 第2-4轮：逐步优化 | 针对问题改进，每次聚焦1-2个维度 |
| 第5轮+：精细打磨 | 最终优化，达到"优秀"标准 |

**每轮迭代必须包含**：
1. 完整的提示词原文
2. AI生成的可视化结果（截图）
3. 对结果的评价分析
4. 下一轮的改进方向

### 三、最终作品要求（30分）

优秀的可视化作品应包含：

| 维度 | 具体要求 |
|------|----------|
| **准确性** | 数据呈现正确，无误导性表述 |
| **清晰性** | 标题明确，轴标签完整，图例清晰 |
| **美观性** | 配色协调，布局合理 |
| **信息量** | 有效传达气候变暖的趋势和关键发现 |
| **专业性** | 注明数据来源，使用规范的术语 |

## 💡 提示词迭代方向参考

你的迭代可以围绕以下维度进行优化：

### 数据维度
- 数据范围（年份、地区）
- 数据聚合方式（年/月/季节）
- 数据对比方式（环比/同比/基线对比）

### 可视化类型
- 折线图 → 面积图 → 热力图 → 地图
- 静态图 → 交互式图表 → 动画

### 设计要素
- 标题和副标题
- 颜色方案（暖色/冷色/渐变）
- 标注和注释（关键事件标记）
- 网格线和背景

### 叙事角度
- 单纯数据呈现
- 强调趋势上升
- 关联历史事件
- 预测未来趋势

## ❓ 常见问题

**Q: 可以使用多个AI工具吗？**
A: 可以，但需要在记录中说明每个工具的使用场景。

**Q: 如果AI无法生成图表怎么办？**
A: 记录失败的尝试本身也是有价值的迭代过程，分析失败原因并调整策略。

**Q: 可以修改数据吗？**
A: 可以选择数据的子集或进行合理的数据处理，但必须说明处理方式。

**Q: 迭代必须线性进行吗？**
A: 不需要，可以尝试不同的方向，记录分支和回溯过程。

**Q: 第1次作业如何评价？**
A: 第1次作业占总成绩10%，由AI初评再复核，强调过程。
